# Repair prompt — Installed Base third-party overclaim

## Use when

A third-party asset is shown with invented connectivity, service coverage, health, telemetry, alert state or evidence.

## Repair instruction

Apply the third-party rule with no exception.

Third-party assets must show:

```txt
Connectivity Unknown
Coverage No Coverage
Health Unknown
Status No record
```

## Remove

```txt
Connected
Premium Service Plan Advanced
Premium Service Plan Essential
Advanced Service Plan
Good health
Excellent health
Critical health
Live telemetry
Active alert
```

when those values are attached to a third-party asset.

## Acceptance

The repaired screen clearly distinguishes third-party inventory visibility from monitored OEM asset intelligence.
