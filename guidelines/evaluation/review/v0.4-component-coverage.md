# v0.4 component documentation coverage

## Purpose

This file verifies that every component introduced or strengthened for v0.4 has clear Figma Make guidance.

This is a review/governance file, not a component usage file.

Component-level guidance remains grouped by functional family:

```txt
guidelines/reference/components/typography.md
guidelines/reference/components/surface-primitives.md
guidelines/reference/components/queue-rows.md
guidelines/reference/components/interactive-panels.md
guidelines/reference/components/navigation-tabs.md
```

This structure is intentional. Figma Make needs to choose by intent and component family, not browse one file per component.
