# v0.5.1 Sprint 3 — GenAI Guideline Refactor Results

## Purpose

Sprint 3 starts the shift from design-system documentation to generation instructions.

Goal:

```txt
Every priority guideline should help a generative AI choose, configure, reject and repair component usage without guessing.
```

This sprint focuses on the components already covered by Storybook in Sprint 2.

---

## Standard introduced

New template:

```txt
guidelines/templates/component-guideline-template.md
```

Required GenAI-ready sections:

```txt
Purpose
Use this component when
Do not use this component when
Prefer this component over
Never generate
Required props
Controlled values
GenAI generation rules
Common generation failures
Repair prompt
Related stories
Related contracts
```

The template is included in the npm package through `package.json > files`.
