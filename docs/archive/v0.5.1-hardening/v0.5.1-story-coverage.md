# v0.5.1 Story Coverage Map

## Purpose

This file records Storybook proof for the `v0.5.1` documentation hardening workstream.

Sprint 2 criterion:

```txt
The documentation must not promise an important component usage that is not demonstrated in Storybook.
```

This file is not a design guideline. It is an audit and coverage map.
