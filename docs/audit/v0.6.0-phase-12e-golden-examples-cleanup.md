# v0.6.0 Phase 12E — Golden examples cleanup report

## Purpose

Phase 12E reduces golden examples to the active v0.6.0 first-generation fixture set and prevents Storybook from importing removed examples.

Golden examples remain test fixtures and Storybook screens, but they should not preserve obsolete domains that are no longer part of active prompts or benchmark contracts.

## Active golden examples

Kept:

```txt
guidelines/examples/golden/customer-monitoring.App.tsx
guidelines/examples/golden/renewal-risk-review.App.tsx
guidelines/examples/golden/asset-recommendation-review.App.tsx
```

These align with:

```txt
benchmarks/figma-make/cases/01-first-generation-customer-monitoring.md
benchmarks/figma-make/cases/02-first-generation-renewal-risk.md
benchmarks/figma-make/cases/03-first-generation-asset-recommendation.md
```

## Removed golden examples

Removed:

```txt
guidelines/examples/golden/qbr-readiness.App.tsx
guidelines/examples/golden/installed-base-explorer.App.tsx
```

Reason:

- the related reusable prompts were removed in Phase 12D;
- the domains are no longer active first-generation benchmark cases;
- keeping them as golden fixtures would preserve a broader v0.5.x example set.

## Storybook impact

Removed:

```txt
src/design-system/stories/golden/installed-base-explorer.stories.tsx
```

Reason:

- it imported the removed `installed-base-explorer.App.tsx` fixture;
- keeping the story would break `npm run build-storybook`.

No `qbr-readiness.stories.tsx` file was found at the expected Storybook golden story path.

## Tests updated

Updated:

```txt
tests/generation-rules/test-targets.ts
tests/generation-rules/golden-examples.test.ts
```

Added:

```txt
tests/generation-rules/golden-storybook-alignment.test.ts
```

The new Storybook alignment test verifies:

- active golden examples exist;
- removed golden examples remain absent;
- Storybook golden stories do not import removed fixtures.

## README updated

Updated:

```txt
guidelines/examples/golden/README.md
```

The README now lists only active v0.6.0 golden examples and explicitly documents that removed examples can be recreated later as v0.7.0 progressive disclosure examples if needed.

## Follow-up

After pulling these changes, run:

```bash
npm run check
npm run build-storybook
```

`npm run check` covers generation-rule alignment. `npm run build-storybook` verifies that no removed golden example is still imported by Storybook.
