## System guidance alignment

`make-minimal-contract.md` and `Guidelines.md` already define the correct separation:

```txt
make-minimal-contract.md → blocking rules
Guidelines.md → full generation router
setup.md → package setup and approved vocabulary
guidelines/prompts/* → reusable generation guidance
benchmarks/figma-make/cases/* → fixed scoring cases
contracts/ → machine-readable rules
```

They also reinforce:

```txt
business patterns before low-level composition
PageHeading over PageHeader
WorkspaceDetailPanel over DetailPanel
ListContainer with queue rows over card stacks
no invented evidence, sources, telemetry, proof or validation
owner, due date and priority for every action
```
