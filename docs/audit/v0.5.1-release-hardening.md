# v0.5.1 Sprint 13 — Release Hardening

## Purpose

Sprint 13 prepares the `0.5.1` patch release.


## Local validation required

Before bumping and publishing, run:

```bash
npm run check
npm run build-storybook
npm pack --dry-run
```

Recommended expanded validation:

```bash
npm run lint:md
npm run test
npm run build:ds
npm run build-storybook
npm pack --dry-run
```

---

## Publish sequence

After validation passes and the version is bumped to `0.5.1`:

```bash
npm publish --tag next
```

Then validate in a consuming app:

```bash
npm install design-system-ai-lab@next
```

Then validate with Figma Make using:

```txt
guidelines/make-minimal-contract.md
guidelines/Guidelines.md
guidelines/prompts/*
guidelines/examples/golden/*
benchmarks/figma-make/cases/*
```

Promote only after validation:

```bash
npm dist-tag add design-system-ai-lab@0.5.1 latest
```
