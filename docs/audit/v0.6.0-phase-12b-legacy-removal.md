# v0.6.0 Phase 12B — Legacy removal report

## Purpose

Phase 12B removes high-confidence legacy files that could keep the project in a mixed v0.5.x / v0.6.0 state.

The project is not in production, so obsolete compatibility files are deleted instead of preserved.

## Decision rule

```txt
Delete legacy when it is not needed by v0.6.0 runtime, reference, evaluation, contracts, tests, benchmarks or source rationale.
```
