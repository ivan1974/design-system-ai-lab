# v0.6.0 Phase 12D — Prompts cleanup report

## Purpose

Phase 12D reduces `guidelines/prompts/` so it no longer behaves like a second runtime documentation system.

The active runtime remains:

```txt
guidelines/Guidelines.md
guidelines/setup.md
guidelines/tokens.md
guidelines/styles.md
guidelines/runtime/*
```

Prompts are now thin reusable task starters that point back to runtime and fixed benchmark cases.

## Kept prompts

The active prompt set is reduced to the three first-generation benchmark domains:

```txt
guidelines/prompts/customer-monitoring.md
guidelines/prompts/renewal-risk-review.md
guidelines/prompts/asset-recommendation-review.md
```

Each kept prompt now:

- states it is reusable guidance, not a fixed benchmark case;
- links to its fixed benchmark case;
- asks Make to read only the v0.6.0 runtime path unless detail is needed;
- imports only from `design-system-ai-lab`;
- imports `design-system-ai-lab/styles.css`;
- routes repair to `guidelines/evaluation/repair/repair-router.md`;
- avoids v0.5.1, `make-minimal-contract.md`, `prompts/workspace-v2.md`, `contracts/components.contract.json` and old `repair-prompts/` paths.

## Removed prompts

The following prompts were deleted:

```txt
guidelines/prompts/qbr-readiness.md
guidelines/prompts/installed-base-explorer.md
guidelines/prompts/overview.md
guidelines/prompts/template.md
guidelines/prompts/workspace-v2.md
```

Reason:

- QBR and installed-base are no longer active benchmark cases.
- `overview.md`, `template.md` and `workspace-v2.md` duplicated runtime guidance.
- Keeping them would reintroduce a parallel prompt system and old v0.5.x reading paths.

## Test update

Updated:

```txt
tests/generation-rules/prompt-system-alignment.test.ts
```

The test now verifies:

- only the three active reusable prompts exist;
- removed prompt files remain absent;
- each active prompt points to its fixed benchmark case;
- each fixed benchmark case points back to its reusable prompt;
- active prompts use v0.6.0 runtime paths;
- active prompts do not contain obsolete v0.5.x vocabulary or paths.

## Decision

`guidelines/prompts/` is kept for now, but only as a small reusable prompt layer.

It is not part of the default package surface and should not replace the runtime files.
